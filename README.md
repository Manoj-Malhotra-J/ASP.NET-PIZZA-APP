# ASP.NET Pizza App 🍕

A Pizza Management Web Application built using ASP.NET Core Razor Pages and C#.

## Features

- View Pizza List
- Add New Pizza
- Delete Pizza
- Responsive UI

## Technologies Used

- ASP.NET Core
- C#
- Razor Pages
- Bootstrap

## Getting Started

```bash
git clone https://github.com/Manoj-Malhotra-J/ASP.NET-PIZZA-APP.git
cd ASP.NET-PIZZA-APP
dotnet restore
dotnet run
```
